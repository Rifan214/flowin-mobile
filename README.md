# Flowin

Flowin adalah aplikasi mobile berbasis Flutter untuk pemesanan dan pengiriman galon air minum. Project ini dibuat sebagai tugas kuliah dengan fokus pada implementasi UI/UX berdasarkan desain yang dibuat menggunakan Google Stitch.

---

## Technology

* Flutter
* Dart
* Material 3

---

## Installation

Clone repository:

```bash
git clone https://github.com/Rifan214/flowin-mobile.git
```

Masuk ke folder project:

```bash
cd flowin-mobile
```

Install dependency:

```bash
flutter pub get
```

Jalankan aplikasi:

```bash
flutter run
```

Menjalankan di Chrome:

```bash
flutter run -d chrome
```
